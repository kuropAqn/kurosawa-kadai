サーバー起動系
・npm　開発環境
npm run build
npm start
^C ※停止

・Apache　本番環境
# 1. プロジェクトディレクトリへ移動
cd ~/kurosawa-kadai

# 2. 最新コードを取得（git を使っている場合）
git pull origin main

# 3. 依存関係の更新（package.json に変更があれば）
npm install
# ── もしくは
yarn install

# 4. 本番用ビルドを作成
npm run build
# ── もしくは
yarn build

# 5. 古いファイルを一掃
sudo rm -rf /var/www/html/*

# 6. 新しいビルド成果物をコピー
sudo cp -r build/* /var/www/html/

# 7. Apache を再起動
sudo systemctl restart httpd
